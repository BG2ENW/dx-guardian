#### 数据库表结构
```sql
CREATE TABLE spot_history (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    callsign TEXT NOT NULL,
    reporter TEXT NOT NULL,
    freq REAL NOT NULL,
    mode TEXT NOT NULL,
    band TEXT,
    comment TEXT,
    grid TEXT,
    lat REAL,
    lon REAL,
    dxcc TEXT,
    cq INTEGER,
    itu INTEGER,
    time TEXT,
    _server_ts REAL NOT NULL,
    source TEXT DEFAULT 'cluster',
    created_at REAL DEFAULT (strftime('%s', 'now'))
);

-- 5 个索引加速查询
CREATE INDEX idx_callsign ON spot_history(callsign);
CREATE INDEX idx_reporter ON spot_history(reporter);
CREATE INDEX idx_time ON spot_history(_server_ts);
CREATE INDEX idx_callsign_time ON spot_history(callsign, _server_ts DESC);
CREATE INDEX idx_reporter_time ON spot_history(reporter, _server_ts);
```

## 已解决问题

### 1. 地图电台标记不显示
**原因**: `spotHistory` 验证逻辑错误 + 字段名不一致（`comment` vs `comments`）  
**解决**: 统一字段名为 `comment`，修复验证逻辑

### 2. 后端收不到我的电台数据
**原因**: JTDX 上报到公共 Cluster，后端连接的是特定 Cluster  
**解决**: 添加 `/api/spot/submit` API，支持直接从 JTDX 上报

### 3. My Spots 面板没有数据
**原因**: 前端只检查内存缓存（500 条），BG2ENW 数据被挤出  
**解决**: 使用数据库查询，获取完整的 7 天数据

### 4. 启动时数据库初始化问题
**原因**: 导入语句重复 + 启动代码未正确添加  
**解决**: 修复导入，重新添加初始化代码

## 关键代码改动

### 后端 (`backend/app.py`)
```python
@app.route('/api/myspots', methods=['GET'])
def api_myspots():
    """返回与指定呼号相关的 Spot 历史"""
    callsign = request.args.get('call', '').upper().strip()
    with lock:
        related = [s for s in spot_history 
                   if s.get('reporter', '').upper() == callsign 
                   or s.get('callsign', '').upper() == callsign]
        i_spotted = [s for s in related if s.get('reporter', '').upper() == callsign]
        they_spotted_me = [s for s in related if s.get('callsign', '').upper() == callsign]
        return jsonify({
            'callsign': callsign,
            'i_spotted': i_spotted,
            'they_spotted_me': they_spotted_me,
            'total_history': len(spot_history)
        })
```
