### 🐛 已修复的 BUG

#### BUG #1: 重复的 QSO 定时器

**问题描述**: 
前端代码中有两个定时器同时调用 `updateQsoStats()`：
- 定时器 A: 每 5 秒调用一次
- 定时器 B: 每 45 秒调用一次

**影响**:
- ⚠️ 每 5 秒就刷新一次 My Spots 面板
- ⚠️ 不必要的网络请求（9 倍冗余）
- ⚠️ 用户体验差（列表频繁刷新）

**修复方案**:
```javascript
// 删除前
setInterval(() => { updateQsoStats(); }, 5000); // 每 5 秒更新
setInterval(() => { updateQsoStats(); }, 45000); // 每 45 秒更新

// 修复后
setInterval(() => { updateQsoStats(); }, 45000); // 只保留 45 秒
```

**文件**: `frontend/js/app.js:748`  
**状态**: ✅ 已修复

def calc_age(ts): return int(time.time() - ts)
def fmt_age(secs):
    if secs < 60: return f"{secs}s"
    elif secs < 3600: return f"{secs // 60}m"
    elif secs < 86400: return f"{secs // 3600}h"
    else: return f"{secs // 86400}d"

for spot in i_spotted:

    spot['age'] = age
    spot['age_formatted'] = fmt_age(age)
